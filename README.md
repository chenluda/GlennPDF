> tesseract 安装：
1. tesseract 下载地址：[https://digi.bib.uni-mannheim.de/tesseract/](https://digi.bib.uni-mannheim.de/tesseract/?C=M;O=D)
2. 将 tesseract 安装路径加入环境变量，如：D:\software\tesseract
3. 安装 pytesseract：
```
pip install pytesseract
```
